type game_over =
| Continue of t * string

val alphabet : string

alphabet is the alphabet allowed for the y axis coordinates of the game board.

val make_coord : int * int -> coord

make_coord (a,b) boxes the coordinate (a,b) into a coord of the same coordinates.

val index : char -> int

index c is the 0-based index of c in the alphabet. Requires: c is an uppercase letter in A..Z.

val inv_index : int -> char

inv_index n takes an index value and returns the corresponding letter in the alphabet. Requires: n is an int between 0-25.

val set_inner : cell list -> int -> cell list

set_inner acc y is a helper function for initializing the empty board. It tail recursively computes a list of Empty cells of length y.

val init_empty_board : int -> int -> cell list list

init_empty_board x y is the function that initializes an empty board of shape (x, y). An empty game board is a 2D array of all Empty cells. Requires: 0 < x < 26, 0 < y < 26

val init_shape : 'a -> 'b -> 'a * 'b

init_shape x y initializes the shape of the game board given a row length of x and column length of y.

val init_player : int -> int -> int -> t

init_player x y name initializes an new player containing an empty board of size (x, y), an empty ship array, and the board's shape (x, y).

val add_ship : ship list -> coord list -> ship list

add_ship ships coords adds a new ship of coordinates coords to the list of current ships for the player: ships. Requires: coords is a valid list of coordinates given the current game board.

val empty_cell : cell -> bool

empty_cell c is the boolean function for whether a given cell c is Empty.

val add_ship_to_board : cell list list -> coord list -> cell list list

add_ship_to_board board coords adds a ship of coordinates coords to the current game board board and returns a new game board. Requires: coords is a valid list of coordinates given the current game board board.

val match_coord : coord -> int * int

match_ coord c is a helper function that extracts the tuple of coordinates from type coord

val get_all_cords : t -> coord -> coord -> coord list
val insert_ship : t -> coord -> coord -> int -> valid_board

insert_ship player start_cord end_cord size inserts a ship given the starting and ending coordinates of the ship (start_cord and end_cord) to the current players board and returns an updated player. If the size does not match the number of coordinates of the ship, then an InavlidB is returned.

val cell_spacing : string

cell_spacing is a constant representing the space between cells on a gameboard. Incorporated to display the game to the user.

val get_dash_space : unit -> string

get_dash_space computes the string of all white space the same size as cell_spacing.

val dash_space : string

dash_spacing is a constant representing the space between cells on a gameboard. The space is the same size as cell_spacing except all white space. Incorporated to display the game to the user.

val hit_val : string
val sunk_val : string
val empty_val : string
val ship_val : string
val parse_cell : cell -> bool -> string

parse_cell c verbose gives the string representation of the cell c. If verbose is true then ships are displayed. Otherwise, they are skipped over and given the empty space.

val get_row : int -> cell list -> bool -> string

get_row row_acc row verbose computes the string representation of row. Prepends the current row label to the row and spaces each cell by the cell_spacing variable. If verbose then the players ships will also be displayed.

val print_row : int -> cell list -> bool -> string

print_row row_acc row verbose prints the string representation of row to the console. If verbose then the players ships will also be displayed.

val print_dash : int -> string

print_dash row_size prints a row of dashes seperated by a dash_space.

val print_top_labels : int -> string

print_top_labels s prints the x-axis coordinates of the game board seperated by cell_spacing. Begins with value of 1 and continues for s coordinate labels.

val print_board : t -> bool -> string

print_board player verbose prints the board of the current player. If verbose then the players ships will also be displayed.

val print_my_board : t -> string

print_my_board player prints the board of the current player showing them their own ships and which spots on the players board has been attacked.

print_my_board t returns a string corresponding to a complete view of t's board

val print_opp_board : t -> string

print_opp_board player prints the opposing players view of the current player's board. This view will hide the players ships from being printed.

print_my_board t returns a string corresponding to a view of t's board without ships

val is_hit : coord -> bool

is_hit c returns true if the coordinate c is a hit.

val is_sunk : coord list -> bool

is_sunk s returns whether ship s is sunk. A ship is considered sunk when all of its coordinates have been hit.

val all_sunk : ship list -> bool

all_sunk s returns whether all ships in s are sunk. A ship is considered sunk when all of its coordinates have been hit. If every ship is sunk then the player has lost the game.

val already_guessed : player -> int * int -> bool

already_guessed board (c1, c2) is the function that determines if cell (c1, c2) has previously been guessed by a user. A cell has not been guessed if its value is Empty or a Ship.

val update_ships : ship list ->
       int * int -> ship list * coord list * cell

update_ships ships (c1, c2) updates a ship list ships according to the guess (c1, c2). If a ship in the list has been hit, then it gets updated and if a ship has not been hit, then there are no changes to the ship list. Returns: tuple of the updated ship list and a list of coordinates to update on the game board. Requires: (c1, c2) has not previously been guessed.

val update_board : cell list list -> coord list -> cell -> board

update_board board update_coords v computes a new board with all instances of update_coords replaced with the cell value v on the current board board.

val check : t -> int * int -> game_over

check p (c1, c2) returns the new player with the coordinate (c1, c2) updated to represent the player's guess and the current player p. Requires: Cell (c1, c2) has not previously been guessed Returns the option of continue or loss
